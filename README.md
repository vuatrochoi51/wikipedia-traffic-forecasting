# Wikipedia Web Traffic Forecasting: ARMA vs SARIMA 📈

This repository contains the source code and documentation for a Time Series Analysis project. The primary objective is to analyze stochastic processes and forecast the daily page views of the Wikipedia article **"Back to the Future"** using ARMA and SARIMA models.

## 🎯 Project Overview
* **Data Source:** Web Traffic Time Series Forecasting from Kaggle.
* **Models Used:** * `ARMA(2,2)` - Autoregressive Moving Average
  * `SARIMA(1,0,1)(1,0,1,7)` - Seasonal ARIMA
* **Goal:** Evaluate the trade-off between minimizing the forecasting error (RMSE) and the model's ability to capture seasonal patterns (7-day cycle).

## 🛠️ Project Structure
* `main.py`: The main Python script that handles data preprocessing, Augmented Dickey-Fuller (ADF) testing, model training, and forecasting.
* `data/`: Contains the `.csv` dataset.
* `images/`: High-resolution outputs (300 DPI) generated by the script, including:
  * Time Series Decomposition
  * ACF & PACF Plots
  * Forecasting comparisons between ARMA and SARIMA.
* `requirements.txt`: Python dependencies.
